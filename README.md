# 🛒 Shopper Spectrum

## Customer Segmentation & Product Recommendation System

Built on real E-Commerce transaction data using RFM Analysis, KMeans Clustering, Collaborative Filtering, and Streamlit.

---

## 📌 Project Overview
This project analyzes 5,41,909 e-commerce transactions to:
- Segment customers into 4 groups using RFM + KMeans Clustering
- Recommend top 5 similar products using Collaborative Filtering

---

## 🚀 Streamlit App Features
- 📦 **Product Recommender** — Enter product name → Get top 5 similar products
- 👥 **Customer Segmentation** — Enter RFM values → Predict customer segment

---

## 🧠 Tech Stack
`Python` `Pandas` `Scikit-Learn` `KMeans` `Cosine Similarity` `Streamlit` `Matplotlib` `Seaborn`

---

## 📊 Customer Segments
| Segment | Description |
|---|---|
| 🟢 High-Value | Recent, frequent, high spenders |
| 🔵 Regular | Steady purchasers |
| 🟠 Occasional | Rare buyers |
| 🔴 At-Risk | Long inactive customers |
